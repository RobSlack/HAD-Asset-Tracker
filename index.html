Connection settings

Paste the three flow URLs from Power Automate here to store your data centrally in SharePoint. Until these are set, the app saves data on this device only.

To get each URL: open the flow → click the first step (When an HTTP request is received) → use the copy icon next to the HTTP URL field. Paste the whole thing, including the long &sig= part at the end.

This app works offline. Changes save on your device instantly and upload to SharePoint automatically when a connection is available. If you've been offline, use Sync now when you're back online.

Anyone who can open this app can see these URLs, and anyone with a URL can run that flow. That's normal for an internal tool, but don't share the app or URLs outside school. If a signature is ever leaked, you can regenerate it in Power Automate and paste the new URL here.
